# DateTimeHeadless

- **Route**: `vue-headless/dist/date-time/component.js`

## Functionalities

- **Multipan**: See more than a month, configure how many and the position of those months.
- **Range selection**: Select a range of years, month or days.
- **Multiple selection**: Select more than one year, month or day.

## API

### Props

| Template         | Script           | Types     | Default      | Description                                           |
| ---------------- | ---------------- | --------- | ------------ | ----------------------------------------------------- |
| `date-view`      | `dateView`       | `Date`    | `new Date()` | Defines the date are the user will view.              |
| `range`          | `range`          | `Boolean` | `false`      | Defines if the calendar will be range-based.          |
| `multiselection` | `multiselection` | `Boolean` | `false`      | Defines if the calendar will be multiselection-based. |
